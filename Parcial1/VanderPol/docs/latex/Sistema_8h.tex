\doxysubsection{Detailed Description}
\mbox{\hyperlink{classSistema}{Sistema}} de dos osciladores de Van der Pol acoplados linealmente. 

Esta clase implementa la simulación, análisis y diagnóstico de un par de osciladores de Van der Pol con acoplamiento lineal.

La dinámica se rige por las ecuaciones\+: \[
\begin{cases}
\ddot{x}_1 - \mu_1(1 - x_1^2)\dot{x}_1 + \omega_{01}^2 x_1 = k(x_2 - x_1), \\
\ddot{x}_2 - \mu_2(1 - x_2^2)\dot{x}_2 + \omega_{02}^2 x_2 = k(x_1 - x_2).
\end{cases}
\]

Incluye herramientas para\+:
\begin{DoxyItemize}
\item Simulación numérica (método RK4)
\item Análisis de sincronización
\item Cálculo del exponente de Lyapunov
\item Validación del paso de tiempo
\item Análisis de bifurcaciones
\end{DoxyItemize}
