\doxysubsection{Detailed Description}
Clase que implementa un oscilador de Van der Pol. 

Este archivo define la clase {\ttfamily \mbox{\hyperlink{classVanDerPol}{Van\+Der\+Pol}}}, la cual modela un oscilador no lineal del tipo Van der Pol descrito por la ecuación diferencial\+: \[
\frac{d^2x}{dt^2} - \mu (1 - x^2) \frac{dx}{dt} + \omega_0^2 x = F_\text{acople}
\] donde\+:
\begin{DoxyItemize}
\item $\mu$ controla la no linealidad y el amortiguamiento.
\item $\omega_0$ es la frecuencia natural.
\item $F_\text{acople}$ representa una fuerza externa o de acoplamiento.
\end{DoxyItemize}

Esta clase se utiliza como componente base del sistema de osciladores acoplados.
