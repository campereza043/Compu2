Calcula las derivadas del oscilador de Van der Pol. 

Calcula las derivadas del sistema de Van der Pol.

Evalúa el sistema de ecuaciones de primer orden equivalente\+: \[
\begin{cases}
\dot{x} = v, \\
\dot{v} = \mu(1 - x^2)v - \omega_0^2 x + F_\text{acople}.
\end{cases}
\]
