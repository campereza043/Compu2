\doxysubsection{Detailed Description}
Define la clase \mbox{\hyperlink{classBola}{Bola}} que representa una partícula (bola) en el sistema de billar. 

Esta clase contiene la posición, velocidad, masa y radio de una bola, así como los métodos necesarios para moverla y resolver colisiones con otras bolas o con las paredes.
